2. Dashboard ESG

Tela principal com indicadores de sustentabilidade.

Adicionar:

Cards de indicadores:

Total de resíduos gerados

Material reaproveitado

Economia gerada

Redução de impacto ambiental

Gráficos:

gráfico de pizza (tipos de materiais)

gráfico de barras (produção de resíduos por setor)

gráfico de linha (geração de resíduos por mês)

3. Registro de Resíduos

Tela para operadores registrarem materiais descartados.

Campos do formulário:

Tipo de material

Peso ou quantidade

Setor de origem

Data

Destino do material

Botões:

Ler código de barras

Registrar material

Adicionar ícone de câmera simulando leitura por código de barras.

4. Rastreamento de Material

Tela para visualizar o fluxo da logística reversa.

Criar um fluxo visual de etapas:

Produção → Separação → Armazenamento → Reaproveitamento ou Descarte.

Mostrar tabela com:

ID do material

tipo

peso

origem

status

destino
